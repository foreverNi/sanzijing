# 三字经儿童故事网页 📖

一个互动式三字经学习网页，为每页三字经配上动画和故事讲解。

## ✨ 功能特点

- 🎬 **197 页图文场景**：每页三字经都配有故事讲解和插图
- 🔊 **语音朗读**：支持调用开源 TTS 服务生成三字经朗读和故事讲解音频
- 📚 **完整内容**：从"人之初，性本善"到"戒之哉，宜勉力"，共 197 页
- 🎨 **儿童友好设计**：柔和的色彩、流畅的动画、直观的翻页
- ↔️ **多种翻页方式**：按钮翻页、键盘方向键、下拉选择页码

## 🚀 快速开始

在线访问：

https://foreverNi.github.io/sanzijing/

直接用浏览器打开 `index.html` 即可使用。

或者启动本地服务器：

```bash
# Python 3
python3 -m http.server 8000

# 或 Node.js
npx serve .
```

然后访问 http://localhost:8000/

## 📁 项目结构

```
├── index.html       # 主页面
```

## 🎯 使用说明

- **翻页**：点击左右箭头 / 按 ← → 键 / 使用底部下拉框选择故事页
- **播放语音**：优先使用 `tts-config.js` 中配置的 TTS 接口；未配置或接口失败时回退到浏览器/系统 TTS
- **数据扩展**：在 `data.js` 中添加新的三字经内容，在 `animations.js` 中定义对应的动画
- **TTS 配置**：参考 `docs/TTS_API.md`，可接入 `server/edge-tts-proxy/` 自托管代理服务
